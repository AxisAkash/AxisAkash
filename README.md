## projects

### OfflineFinanceManager

local-first personal finance tracker for the Bangladesh market

| | |
|---|---|
| **what** | React Native + Expo app managing finances entirely offline via SQLite. Biometric auth, transaction tracking, category management, receipt sharing via push notifications. |
| **architecture** | offline-first with local database, designed for unreliable connectivity |
| **status** | active development |

[**source**](https://github.com/AxisAkash/OfflineFinanceManager)

---

### Car Rent Management

full-featured car rental platform with admin dashboard

| | |
|---|---|
| **what** | PHP MVC system with MySQL backend. User/admin auth, car CRUD, order/invoice/payment management, rental history, blog module. Chart.js dashboard analytics. |
| **architecture** | server-rendered MVC with PDO prepared statements |
| **status** | complete |

[**live demo**](https://car-rental-system.fwh.is/?i=1) &nbsp;·&nbsp; [**source**](https://github.com/AxisAkash/Car-Rent-Management)
