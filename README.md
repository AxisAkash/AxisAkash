## projects

**OfflineFinanceManager** — local-first personal finance tracker for Bangladesh. React Native, Expo, SQLite, biometric auth, offline-first architecture. *active development*

**Car Rent Management** — full-featured PHP MVC platform with MySQL. User/admin auth, car CRUD, order management, invoices, payments, rental tracking, Chart.js dashboard.

**Digital Currency Converter** — PWA with live exchange rates, offline support, glassmorphic UI. Vanilla JavaScript, no frameworks.

**Ai Calculator** — multi-mode calculator with standard, scientific, converter, and AI-assisted solving modes. PWA-ready.

[source](https://github.com/AxisAkash/Ai_Calculator) · [live](https://AxisAkash.github.io/Ai_Calculator)

**Student Manager** — browser-based CRUD with attendance tracking, dark mode, search/filter, CSV export. LocalStorage-backed SPA.
