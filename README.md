## Featured Projects

### OfflineFinanceManager

_A local-first personal finance tracker for Bangladesh_

Built with React Native and Expo, this app manages personal finances entirely offline using SQLite. Features include biometric authentication, transaction tracking, category management, and receipt sharing. Designed for users without reliable internet access.

|                  |                                                                        |
| ---------------- | ---------------------------------------------------------------------- |
| **Built with**   | React Native, Expo, TypeScript, SQLite                                 |
| **Architecture** | Local-first, offline-capable with biometric auth                       |
| **Key features** | Transaction CRUD, category management, push notifications, CSV sharing |
| **Status**       | Active development                                                     |

[**Source code** →](https://github.com/AxisAkash/OfflineFinanceManager)

### Digital Currency Converter

_A PWA currency converter with live exchange rates_

Progressive web app featuring real-time currency conversion, conversion history, and offline support. Clean UI with glassmorphic design principles. No build tools — pure vanilla JavaScript.

|                  |                                                          |
| ---------------- | -------------------------------------------------------- |

| **Architecture** | PWA with service worker, vanilla JS frontend             |
| **Key features** | Live rates, conversion history, offline mode, CSV export |
| **Status**       | Complete                                                 |
